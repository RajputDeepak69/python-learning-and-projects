# 🔍 GitHub PR Fetcher (Python)

## 👨‍💻 About This Project
I am currently learning Python and building small practical projects to understand how APIs and automation work.

This script fetches data from a public GitHub repository API and prints the usernames of people who created pull requests.

## ⚙️ How It Works
- User provides a GitHub API URL
- Script sends a GET request using `requests`
- Converts response into JSON
- Loops through data and prints PR creator usernames

## 🧠 Concepts Used
- Python `requests` library
- Working with APIs
- JSON parsing
- Loops and indexing
