# 📂 List Files from Folder (Python Script)

## 👨‍💻 About This Project
I am currently **new to Python** and actively learning scripting and automation.  
This project is part of my hands-on practice to understand how Python can be used for real-world tasks.

## 🧠 What I Learned
- Basics of Python scripting
- Using `os` module to interact with the system
- Looping through user input
- Exception handling (`try-except`)
- Handling invalid folder names gracefully

## ⚙️ How It Works
- User enters folder names (space separated)
- Script loops through each folder
- Lists all files inside
- Handles errors if folder doesn't exist
